# DDG Search Skill

Search the web using DuckDuckGo without needing an API key.

## Usage

```
/ddg-search "your search query"
```

## Features

- No API key required
- Uses DuckDuckGo search engine
- Returns structured results with titles, URLs, and descriptions
- Lightweight and fast

## Examples

- `/ddg-search "Claude AI"`
- `/ddg-search "latest news in tech"`
- `/ddg-search "how to learn Python"`
